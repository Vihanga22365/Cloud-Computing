# Cloud-Computing

* Download Zip File
* Extract the Zip File
* Open that project using VS Code 
* run <code> cd .\api\ </code>
* run <code> npm install </code>
* run <code> cd .. </code>
* run <code> cd .\client\ </code>
* run <code> npm install </code>
* run <code> cd .. </code>
* run <code> docker-compose build </code>
* Open Browser
* run <code> http://localhost:3000/ </code>
